# Heartland Website Tools

Use this skill to improve the Heartland Medical Transportation website.

Focus on adding the essential tools now:

1. Request a Ride form
2. Careers application form for EMT / EVC
3. Email notifications to admin@emsopshub.com
4. Database storage for ride requests and job applications
5. Resume upload for applicants
6. Service area map
7. SEO and social preview setup
8. Analytics for calls, ride requests, and job applications
9. Spam protection and form validation
10. Fast mobile-first performance

Recommended free-tier tools:
- Supabase for database and file storage
- Resend for email notifications
- Cloudflare for DNS, SSL, CDN, and protection
- Google Maps or OpenStreetMap for maps
- Google Analytics or Plausible for analytics
- Cloudinary for image optimization if needed

Design must stay premium, cinematic, black and gold, with glassmorphism and luxury medical branding.

Do not overcomplicate the website. Prioritize phone calls, ride requests, job applicants, trust, and professional credibility.
